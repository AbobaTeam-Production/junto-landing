Что внутри

Один плеер — на всех.

Никакого «нажмите паузу на счёт три». Junto держит время и видеопоток в синхроне для всей комнаты — а сверху добавляет всё, что делает совместный просмотр живым.

Кадр в кадр.

WebSocket-синхронизация под < 120 мс. Кто-то нажал паузу — пауза у всех, в ту же секунду. Перемотка тоже общая.

Голос без эха.

WebRTC-канал с автоматическим заглушением источника, когда кто-то говорит. Слышите друг друга, не фильм через друг друга.

Хочешь свой инстанс — разверни у себя.

Backend под AGPL, фронт под MIT. Свой VPS, свой LiveKit, свой контроль трафика — без ограничений тарифа. Self-hoster получает все Pro-возможности по умолчанию.

Открыть на GitHub junto-frontend junto-backend 
# clone & bring up the stack
git clone https://github.com/AbobaTeam-Production/junto-backend
cd junto-backend
cp .env.example .env  # LIVEKIT_KEYS, TMDB_*, FCM_*
docker compose up -d  # django + postgres + redis + livekit

# point your DNS:
api.example.tld      → 127.0.0.1:8000

Частые вопросы

Кратко и по делу.

Сколько это стоит?
Базовый совместный просмотр — бесплатно: до 2 гостей в комнате, голос, синхрон, история 30 дней. Junto Pro — 199 ₽/мес (1 990 ₽/год): до 10 гостей, без рекламы, без лимита часов, история навсегда. Cinema — 499 ₽/мес: до 25 гостей, темы, watch-party события. Подробнее в тарифах.
Можно развернуть у себя на сервере?
Да. Backend и frontend на GitHub под открытыми лицензиями. Поднимаете docker-compose с Postgres / Redis / LiveKit, прописываете DNS — и получаете полный аналог managed-инстанса. Self-host автоматически включает все Pro-возможности. Инструкция в блоке Self-host.
Откуда стримятся фильмы и есть ли пиратство?
Junto не хостит контент. Это плеер с синхронизацией: вы приносите файл с диска, magnet-ссылку или URL — приложение лишь проигрывает. Ответственность за источники остаётся на пользователях.
Сколько человек помещается в комнату?
Сейчас комфортно до 8 человек одновременно с голосом и видеопотоком. Без голоса — больше. Лимит мы держим, чтобы голосовой канал не хрипел на слабых соединениях.
Работает на старых телефонах?
Минимум — Android 8.0. На устройствах 2018+ годов идёт без проблем. На совсем старых может тормозить декодинг 4K, но 1080p тянут почти все.
А если у нас разный интернет?
Junto адаптивно подстраивает битрейт под самого медленного в комнате — лучше пусть все смотрят 720p синхронно, чем кто-то отстанет. По желанию можно отключить адаптацию в настройках.
Где исходники и как помочь?
Репозиторий — на GitHub. Pull request'ы, баги и идеи приветствуются. Особенно нужны переводчики и тестировщики на iOS.
